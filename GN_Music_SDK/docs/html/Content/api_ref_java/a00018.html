Detailed Description

This class processes an audio stream for recognition.

The input stream is received via the writeBytes method. Results are returned

via a GNResultReady object.

Starts the stream processing session by initializing objects needed

to allow writeBytes to continuously feed audio samples to a buffer.

Note that no recognition is performed until idNow() is called.

Writes PCM audio data to the GNRecognizeStream object. The PCM data should be of the

same format specified in the GNAudioConfig object. Buffers of consistent length representing

less than a second of PCM data are best.
