Detailed Description

Describes the status of an operation as it progresses.

GNStatus is provided to the application via the GNOperationStatusChanged interface. It

provides a detailed description of the operation's current state and progress.

int com.gracenote.mmid.MobileSDK.GNStatus.getPercentDone ( )

Obtains the completion percentage of the associated operation.

Operations performing a deterministic function report the function's completion percentage;

for example, recording audio from a microphone.

For non-deterministic functions, the completion percentage is not known; in this

case, this function always returns zero.

Returns
Completion percentage of the associated operation's current function
