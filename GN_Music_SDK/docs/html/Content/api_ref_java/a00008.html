Detailed Description

The GNAudioSourceMic class manages resources for applications to stream audio from

the platform's audio input hardware. The application receives the recorded audio (PCM-encoded)

by implementing the GNAudioSourceMicDelegate interface.

com.gracenote.mmid.MobileSDK.GNAudioSourceMic.GNAudioSourceMic ( GNAudioConfig  audioConfig,
GNAudioSourceDelegate  inAudioSourceMicDelegate 
)

Acquires audio input hardware resource and initializes it as specified by audioConfig.

Below are the limitations to the audio configuration parameters supported:

  • number of channels: 1 or 2, otherwise the platform's default channel configuration is used

  • bytes per sample: 1 or 2, otherwise 2 is used by default

Parameters
audioConfig
                            configuration object specifying audio data format
inAudioSourceMicDelegate
